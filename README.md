---

## 🌊 Core Operations

### 1. Hull Maintenance (Hydrodynamics)
* **The Problem:** Biofouling (barnacles/algae) increases fuel consumption by up to **20%**.
* **The Solution:** Use hydraulic **Brush Karts** that suction to the hull, stripping growth without damaging anti-fouling paint.

### 2. Propeller Polishing (Mirror Finish)
* **The Goal:** Achieve **Rubert Grade A** finish to stop **Cavitation** (imploding bubbles that erode metal).
* **The Result:** Smoother thrust, less vibration, and maximum fuel efficiency.

### 3. Precision Metrology (Wear Monitoring)

| Tool | Application | Purpose |
| :--- | :--- | :--- |
| **Feeler Gauge** | Rudder / Wear Rings | Measuring 0.05mm – 2.0mm clearance. |
| **Poker Gauge** | Tailshaft Bearing | Measuring shaft "drop" against baseline. |

### 4. Pressure Management (Sensor Swaps)
* **The Seal:** Use **Cofferdams** or wood plugs with plastercine to **"plug the ship."**
* **The Protocol:** Strict communication and pressure equalization before detaching equipment to ensure safety.
